- git-lfs 설치 (필수, 기존의 git이 있더라도, 큰 용량을 위한 git)

<pre><code>
    sudo apt update
    sudo apt install git-lfs
</code></pre>

- 모델 다운로드
<pre><code>

    // mkdir my_model
    // cd my_model

    git clone https://huggingface.co/meta-llama/Meta-Llama-3.1-8B-Instruct
</code></pre>

- 허깅 페이스 아이디 입력 요구

- 허깅 페이스 접속 토큰 요구
